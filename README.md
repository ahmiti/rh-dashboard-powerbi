# RH Dashboard – Power BI

## Description
Ce projet présente un tableau de bord RH interactif développé avec Power BI afin d’analyser les données des employés et faciliter la prise de décision.

## Objectifs
- Suivi des indicateurs RH
- Analyse des effectifs
- Visualisation des performances

## Technologies
- Power BI
- Power Query
- Excel / SQL

## Fonctionnalités
- KPIs dynamiques
- Filtres interactifs
- Graphiques analytiques

## Architecture
Data Source → Power Query → Data Model → Dashboard
